Getting Started with EdgeSync

Welcome to EdgeSync, the comprehensive multi-tenant IoT platform that empowers you to sync, visualize, and act on your data in real-time. This guide will walk you through the essential steps to get your IoT project up and running.

Platform Overview

EdgeSync is a cloud-based IoT platform designed for data and device management, featuring:

  • Cloud-based MQTT broker management - Secure and scalable messaging
  • Edge device registration - Simple device onboarding process
  • Real-time data logging - Efficient data collection and storage
  • Edge analytics - Process and analyze data at the edge
  • EdgeBoard designer - Low-code dashboard creation tool
  • Over-the-air updates - Keep devices current remotely
  • Natural language automation - Intelligent workflow integration
Note: EdgeSync leverages AWS IoT platform and state-of-the-art architecture for remarkable scalability and cost efficiency.

Account Setup

To begin using EdgeSync, you'll need to create an account and configure your initial settings.

Creating Your Account

  1. Visit the EdgeSync homepage and click "Sign Up"
  2. Choose your preferred authentication method:
    • Google OAuth (recommended for quick setup)
    • Email and password registration
  3. Complete the verification process
  4. Access your dashboard

Initial Configuration

Once logged in, you'll be guided through the initial setup process:

  • Organization Setup: Define your organization details
  • Project Creation: Create your first IoT project
  • Security Settings: Configure access controls and permissions
// Example: Initial API configuration
const edgeSync = new EdgeSyncClient({
    apiKey: 'your-api-key',
    region: 'us-east-1',
    organization: 'your-org-id'
});

Device Registration

EdgeSync makes device registration simple and secure. Follow these steps to register your first device:

Device Types

EdgeSync supports various device types:

  • ESP32 devices - Full OTA update support
  • Industrial PLCs - Enhanced edge capabilities
  • Custom IoT devices - Flexible integration options
  • Sensors and actuators - Real-time data collection

Registration Process

  1. Navigate to the "Devices" section in your dashboard
  2. Click "Add New Device"
  3. Select your device type and configuration
  4. Generate device credentials
  5. Configure device connection settings
Security Note: Always store device credentials securely and rotate them regularly for enhanced security.

Device Configuration

// Example: Device configuration for ESP32
{
    "deviceId": "esp32-sensor-001",
    "deviceType": "ESP32",
    "mqttEndpoint": "your-mqtt-broker.edgesync.com",
    "certificates": {
        "deviceCert": "-----BEGIN CERTIFICATE-----...",
        "privateKey": "-----BEGIN PRIVATE KEY-----...",
        "caCert": "-----BEGIN CERTIFICATE-----..."
    },
    "topics": {
        "telemetry": "devices/esp32-sensor-001/telemetry",
        "commands": "devices/esp32-sensor-001/commands"
    }
}

Dashboard Creation

The EdgeBoard designer allows you to create stunning dashboards with a low-code approach.

Dashboard Components

  • Data Widgets: Charts, graphs, and gauges
  • Control Panels: Buttons, sliders, and switches
  • Status Indicators: LED indicators and status badges
  • Custom Components: Build your own widgets

Creating Your First Dashboard

  1. Access the EdgeBoard designer from your project dashboard
  2. Choose a template or start from scratch
  3. Drag and drop components onto the canvas
  4. Configure data sources and bindings
  5. Customize styling and layout
  6. Publish your dashboard

Data Binding
